Domain names and ICANN

A Domain Name is an address that a user can type into their web browser to get to the desired website. Much like how a person has their own phone number which you can call to reach them, a website has its own web address, commonly known as its domain name. In addition, a website can have infinitely many domain names, similarly to how a single person can have many different phone numbers. You only need to know one of these domain names to reach the website you want to visit.

The domain name for this webpage is web.njit.edu. The following text (/~nak29/internet/domain.html) is another section of the web address that directs you to a specific page within the site. The important part of the domain name is everything after the “www.” and before the first forward slash.

The commonly used “.com” and “.org” suffixes help identify the type of website the domain name represents. The “.com” suffix is most commonly used because it’s typically used for commercial websites. Following the same pattern, the “.org” suffix is used to represent a non-profit organization’s website, and the “.gov” suffix represents government websites.

ICANN, the Internet Corporation for Assigned Names and Numbers, is a non-profit organization that deals with ensuring all users of the Internet can find all valid addresses. ICANN works to keep the operational stability on the internet in balance.
